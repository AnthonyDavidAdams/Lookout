Setup

Up and running in two minutes.

One command installs everything. No dependencies, no complexity.

Clone

Clone the repo from GitHub to your Mac.

Install

Run ./install.sh and it builds and installs automatically.

Authorize

Add your Claude API key and grant Screen Recording permission.

Ask

Click the eye in your menu bar and start asking about your screen.
